# 식물의 잎 분류하기 (Plant Leaf Classification)

초등학교 3학년 과학 **「4. 식물의 생활」** 단원용 디지털 분류 도구입니다.
잎 카드를 분류 기준(질문)에 따라 **그렇다 / 그렇지 않다** 두 칸으로 드래그하며 식물을 분류합니다.

## 기능
- 잎 카드 드래그 앤 드롭 분류
- 분류 기준 질문 선택 또는 직접 입력 (실시간 반영)
- 분류 결과 새로고침 (식물 목록은 유지)
- 모두 분류하면 폭죽 + 효과음
- 브라우저 로컬스토리지에 분류 상태 자동 저장
